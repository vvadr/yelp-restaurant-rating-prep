# Module 3 Class Notebooks

These notebooks document the full Yelp Restaurants Module 3 lab sequence.

Run order:

1. `01_class1_data_cleaning.ipynb`
2. `02_class2_encoding_scaling.ipynb`
3. `03_class3_feature_engineering.ipynb`
4. `04_class4_feature_selection.ipynb`
5. `05_class5_pipelines.ipynb`
6. `06_class6_end_to_end_lab.ipynb`

The notebooks use `data/processed/yelp_restaurants_clean.parquet`, which is generated by:

```powershell
python scripts/run_pipeline.py
```

Raw Yelp JSON files are not included in the repository because they are too large for GitHub.
